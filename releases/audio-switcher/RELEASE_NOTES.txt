# Audio Device Switcher v1.0.0

Portable Windows audio output switcher by MPTech Tools.

## Recommended download

Download the portable version:

**MPTech-Audio-Switcher-v1.0.0-portable.exe**

The portable version does not require installation.

If included, the installer version is optional:

**MPTech-Audio-Switcher-v1.0.0-installer.exe**

## Notes

Windows SmartScreen may show a warning because the executable is not code-signed yet.
